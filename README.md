# UppalCRM Marketing Website

A modern, responsive marketing website for UppalCRM - a B2C CRM software designed specifically for software licensing businesses.

## Features

- **Modern Design**: Clean, professional design with smooth animations and hover effects
- **Mobile Responsive**: Fully responsive design that works on all devices
- **Hero Section**: Compelling hero with interactive dashboard preview
- **Features Section**: Highlights contact-centric approach vs traditional B2B CRMs
- **Pricing Tiers**: Four pricing plans (Starter $49, Professional $149, Business $299, Enterprise custom)
- **Trial Signup Form**: Comprehensive form capturing company information
- **Interactive Elements**: Smooth scrolling, form validation, and loading states
- **Analytics Ready**: Built-in tracking for conversions and user interactions

## Getting Started

1. **Clone or Download**: Get all files in the same directory
2. **Open in Browser**: Simply open `index.html` in any modern web browser
3. **Deploy**: Upload all files to your web hosting provider

## Deployment Options

### Static Hosting (Recommended)
- **Netlify**: Drag and drop the folder to deploy instantly
- **Vercel**: Connect to GitHub for automatic deployments  
- **GitHub Pages**: Host directly from a GitHub repository
- **AWS S3**: Simple static website hosting
- **Cloudflare Pages**: Fast global CDN with automatic builds

### Traditional Hosting
- Upload all files to your web hosting provider's public_html directory
- Ensure `index.html` is in the root directory

## Customization

### Colors and Branding
- Update CSS custom properties in `:root` section of `styles.css`
- Replace "UppalCRM" with your brand name throughout files
- Update favicon and add your logo

### Form Integration
- Replace the simulated API call in `script.js` with your actual endpoint
- Integrate with email services (Mailchimp, SendGrid, etc.)
- Add CRM integration for lead capture

### Analytics Integration
- Add Google Analytics tracking code
- Integrate Facebook Pixel for ads
- Connect to your preferred analytics platform
